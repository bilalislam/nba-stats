## Requirements
- You will need go 1.16 +

## How to use
ps: total 4 minutes wait and all match will finalized.

```sh
$ go build *.go
$ ./main
```

## How to use with docker
```sh
$ docker build -t nba-stats . 
$ docker run nba-stats
```
